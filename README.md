Project Name: LasustechLMS (Library Management System) Version: 1.0.0 Technology Stack: HTML5, Tailwind CSS, Vanilla JavaScript (ES6+)

Live Application URL: https://lasustechlms.onrender.com/

Github repository: https://github.com/MichaelOlawale/LasustechLMS

**Summary**

LasustechLMS is a web-based Single Page Application (SPA) designed to digitize the library operations of Lagos State University of Science and Technology. It serves two distinct user groups: Students and Librarians. The application facilitates the searching, borrowing, returning, and management of library resources through a responsive, user-friendly interface.

Unlike traditional systems requiring complex backend setups, this application utilizes Browser LocalStorage for data persistence, making it lightweight, portable, and easy to deploy without server-side dependencies.


✨ **Key Features**

Role-Based Access Control: Distinct, customized dashboards tailored for Students (Members) and Librarians (Admins).

Interactive Digital Catalog: A live inventory where users can search for books by title, author, or ISBN, filter by category, and instantly see how many copies are available.

Automated Circulation System: A seamless digital borrowing and returning mechanism that instantly updates the library's inventory counts in real-time.

Personalized Student Dashboard: A dedicated space for students to monitor their active borrowed books, track upcoming due dates, and manage their reading history.

Data Persistence: Utilizes the browser's LocalStorage to save data, ensuring that changes (like newly added books or active loans) are maintained across sessions.

Modern UI/UX: Fully responsive design that works perfectly on both mobile phones and desktop computers, featuring an integrated Dark Mode toggle for comfortable reading.
